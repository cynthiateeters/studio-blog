# Contributing to HyBit Studio

Thank you for your interest in improving HyBit Studio! This blog shares practical insights on AI-augmented web development curriculum, and reader feedback helps keep it accurate and useful.

## How to contribute

### Found an error?

If you spot incorrect information, outdated instructions, or broken links:

1. [Open a Correction issue](../../issues/new?template=correction.md)
2. Include the post title or URL
3. Describe what's wrong and (if you know) what the fix should be

### Have a question?

If something in a post is unclear or you'd like more detail:

1. [Open a Question issue](../../issues/new?template=question.md)
2. Reference the relevant post
3. Share what you've already tried

### General feedback

For broader discussions, suggestions, or just to say hello, [connect with me on LinkedIn](https://www.linkedin.com/in/cynthia-teeters/).

## What I'm not looking for

- Pull requests for blog content (this is a personal blog)
- Feature requests for the website itself
- Spam or self-promotion

## Code of conduct

Be kind, be constructive, be helpful. We're all here to learn.
